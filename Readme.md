# Initialise Python Environment :

Run the following :

    python -m venv venv
    source venv/bin/activate

# Install Dependencies :

Run The Following :

    pip install -r requriments

# Run App
    python src/app.py

# Setup Sqlite Database :

Run the following :
Initialise alembic :

    alembic init alembic
    alembic upgrade head

# Running tests:
    pytest
